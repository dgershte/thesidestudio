Storyboard Differencer

My last work term I spent at an Ottawa-based startup - Crank Software - where I spent most of my time working on a merging tool that differences and merges two Storyboard models. My goal was to automate as much of the differencing as possible, and let the user visually select which differences or updates they would like to bring over from one model to the other. The most difficult part was trying to figure out a good balance between which procedures should be automated and which should be up to the user to decide. Considering this was a whole new area of the Storyboard Suite, it was fun to get a whole project running from scratch.

Gradients

A simple gradient generator, using only html div's and CSS styles.
Add Black:Adds a black stop to the center of the gradient.
Add White:Adds a white stop to the center of the gradient.
Pop:removes and returns the last stop.
Shift:removes and returns the first stop.
Reverse: Reverse the order of all stops.
Sort: Sorts the stops.
Map:Modifies the gradient by calling the callback for every stop.
Clear: Clears the whole gradient

Maze

A quick Maze path-finding algorithm I wrote while procrastinating studing for exams a few terms ago. As the user clicks on the tiles, an Adjacency matrix is created. Once the bottom-lefct corner is selected, a depth-first search algorithm is applied to the adjacency matrix and the maze is solved. Begin building the maze by selecting any of the outer tiles (other than the bottom right corener) and once you click on the bottom right corner, the shortest-distance finder will execute and solve the maze.
